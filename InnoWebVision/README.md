# InnoWebVision - Hotel Website Redesigns

A modern, responsive website for InnoWebVision - a service specializing in website redesigns for independent US hotels to increase direct bookings and improve mobile experience.

## Project Structure

```
InnoWebVision/
├── index.html          # Main HTML file
├── css/
│   └── styles.css      # Stylesheet
├── images/             # Portfolio images (placeholder)
├── README.md           # This file
└── .github/
    └── copilot-instructions.md
```

## Features

- **Responsive Design**: Mobile-first, works on all devices
- **Modern UI**: Clean, professional interface with smooth transitions
- **Portfolio Section**: Showcase of concept redesigns for different hotel types
- **Call-to-Action**: Clear buttons and contact information
- **Performance Optimized**: Lightweight CSS with optimized styling

## Getting Started

1. Open `index.html` in a web browser to view the website
2. All styling is contained in `css/styles.css`
3. Replace placeholder images in the `images/` folder with actual portfolio images

## Image Placeholders

The following image files need to be added to the `images/` folder:
- `boutique-before.png` - Before redesign screenshot
- `boutique-after-desktop.png` - After redesign (desktop)
- `boutique-after-mobile.png` - After redesign (mobile)
